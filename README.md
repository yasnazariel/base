# Base

Base is an Ethereum Layer 2 (L2) rollup designed to provide fast, low-cost, and secure onchain experiences for developers and users.

## Why Base

- **Low cost and high speed:** Base enables near-instant transactions with fees typically under one cent.
- **Built for global access:** Anyone can use Base to build and interact with onchain applications.
- **Developer growth:** Base supports builders through grants, distribution, and ecosystem support.
- **Powerful tooling:** Access a wide range of tools for building applications across AI, social, media, and entertainment.

## Learn More

- Visit the [docs](https://docs.base.org) to:
  - [Connect your wallet](https://docs.base.org/base-chain/quickstart/connecting-to-base)
  - [Run a node](https://docs.base.org/base-chain/node-operators/run-a-base-node)
  - [Deploy an app](https://docs.base.org/base-chain/quickstart/deploy-on-base)

- Visit the [specs](https://specs.base.org) site for protocol details, including past and upcoming upgrades.

## Install Binaries

You can install prebuilt binaries using [`baseup`](baseup/README.md):

```bash
curl -fsSL https://raw.githubusercontent.com/base/base/main/baseup/install | bash
```

This script downloads and installs the latest release binaries for Base.
